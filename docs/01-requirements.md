# Project Requirements

## Project Name

Nike Inspired E-Commerce Store

## Goal

Build a complete MERN stack e-commerce platform where users can browse products, add products to cart, place orders, and manage their accounts.

## User Features

- Register Account
- Login
- Browse Products
- Search Products
- View Product Details
- Add To Cart
- Manage Cart
- Add To Wishlist
- Checkout
- View Orders
- Manage Profile

## Admin Features

- Login
- Add Product
- Update Product
- Delete Product
- Manage Inventory
- View Orders
- Update Order Status

## Future Features

- Stripe Payment
- Reviews and Ratings
- Product Recommendations
- Email Notifications
