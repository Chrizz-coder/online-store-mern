# API Design

## Authentication

Purpose:
Manage user accounts and login sessions.

Actions:

* Register User
* Login User
* Logout User
* Get Current Logged In User
* Update User Profile

---

## Products

Purpose:
Manage product information.

Customer Actions:

* View All Products
* View Single Product
* Search Products

Admin Actions:

* Create Product
* Update Product
* Delete Product

---

## Cart

Purpose:
Manage products selected for purchase.

Actions:

* View Cart
* Add Product To Cart
* Remove Product From Cart
* Update Product Quantity
* Clear Cart

---

## Orders

Purpose:
Manage purchases.

Customer Actions:

* Create Order
* View Orders
* View Single Order

Admin Actions:

* View All Orders
* Update Order Status

---

## Admin

Purpose:
Store management.

Actions:

* Manage Products
* Manage Orders
* View Users
* Manage Inventory
